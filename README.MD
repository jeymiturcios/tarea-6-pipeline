# Tarea 6 - Pipeline de Integración Continua

![CI Status](https://github.com/jeymiturcios/tarea-6-pipeline.git)

## Descripción
Este proyecto implementa un pipeline de CI con GitHub Actions para validar funciones matemáticas básicas en Node.js.

## Cómo ejecutar localmente
1. Instalar dependencias: `npm install`
2. Correr pruebas: `npm test`
3. Correr linter: `npm run lint`
